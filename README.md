# Tabuada

 Construir uma tabuada sem frameworks para fixar o aprendizado base de JS, HTML5 e CSS3. (Aula CursoEmVideo)

 - Criar elemento dinâmico HTML
 - Iniciar aplicação com o cursor no Input
 - Limpar Input e Resposta
 - Variavel.Value

#### HTML5
 - Select e Option
 - Footer
 - Centralizar

#### CSS3
 - Sombra
 - Margin
